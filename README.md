# IOE-Hub

> **An all-in-one academic companion for Institute of Engineering (IOE) students.**

IOE-Hub is a full-stack web application designed to help IOE students organize and manage their academic life from a single platform. It brings together essential student tools such as assignment tracking, semester planning, GPA calculation, notes management, and reminders to make university life more organized and productive.

The goal of IOE-Hub is to provide a centralized platform that grows alongside a student's academic journey.
